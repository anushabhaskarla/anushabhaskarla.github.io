PROJECT BACKGROUND

In one of my courses, I was tasked with creating the user interface for an internal Starbucks store's database. With the user of a Starbucks store employee in mind, I created the above interface following a simplistic design. The interface is in the form of a website, and displays a 'sneak peak' of the features a full fledged interface of this nature would have. Click the photograph above to explore the interface of this project.

Key Features: Add, delete, and edit functionality within tables, Data/Table Directory, Link to external website, "Featured Tables"

Course: Database Design & Management

Tools: HTML/CSS/JavaScript
